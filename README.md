# Silva & Advogados

Este é o repositório do site institucional do escritório **Silva & Advogados**, especializado em Direito Civil, Empresarial, Trabalhista e de Família. O projeto tem como objetivo apresentar os serviços, a equipe, os diferenciais do escritório, e fornecer um canal de contato eficiente para os clientes.

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido com as seguintes tecnologias e bibliotecas:

- **[React 19](https://react.dev/)**: Biblioteca JavaScript para construção da interface de usuário.
- **[Vite](https://vitejs.dev/)**: Ferramenta de build rápida e otimizada.
- **[Tailwind CSS v4](https://tailwindcss.com/)**: Framework utilitário de CSS para estilização ágil e responsiva.
- **[Framer Motion](https://www.framer.com/motion/)**: Biblioteca para criar animações fluidas e interativas.
- **[React Router Dom](https://reactrouter.com/)**: Gerenciamento de rotas da aplicação (Navegação SPA).
- **[Lucide React](https://lucide.dev/)**: Ícones elegantes e consistentes.

## 📦 Como executar o projeto localmente

Siga os passos abaixo para rodar o projeto em sua máquina:

1. Clone o repositório:
   ```bash
   git clone <URL_DO_REPOSITORIO>
   ```

2. Acesse o diretório do projeto:
   ```bash
   cd site-adv-2
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```

4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

5. O servidor iniciará localmente. Abra no navegador (por padrão em `http://localhost:5173/`).

## 🛠 Scripts Disponíveis

No diretório do projeto, você pode rodar:

- `npm run dev`: Inicia a aplicação em modo de desenvolvimento.
- `npm run build`: Cria a build de produção otimizada na pasta `dist`.
- `npm run lint`: Executa o ESLint para encontrar problemas no código.
- `npm run preview`: Pré-visualiza a build de produção localmente.
